# Extension Icons

This directory contains the extension icons in various sizes.

## Required Sizes

- `icon-16.png` - 16x16px (toolbar)
- `icon-32.png` - 32x32px (toolbar retina)
- `icon-48.png` - 48x48px (extension management page)
- `icon-128.png` - 128x128px (Chrome Web Store, notifications)

## Creating Icons

You can create icons using:
1. **Online tools:** favicon.io, canva.com
2. **Design software:** Figma, Adobe Illustrator, Sketch
3. **Icon generators:** realfavicongenerator.net
