# Alert Sound

This directory contains the alert sound played when a meeting reminder fires.

## Required File

- `alert.mp3` - The notification sound

## Sound Requirements

- **Format:** MP3 (best compatibility)
- **Duration:** 1-3 seconds
- **Volume:** Moderate (not too loud)
- **Tone:** Pleasant but attention-grabbing

## Where to Get Sounds

### Free Sound Libraries

1. **Freesound.org** - https://freesound.org/
   - Search: "notification", "alert", "bell"
   - License: CC0 or CC-BY

2. **Zapsplat** - https://www.zapsplat.com/
   - Free sound effects library

3. **Mixkit** - https://mixkit.co/free-sound-effects/
   - Free notification sounds

### Create Your Own

- Use online tone generators
- Record and edit in Audacity (free)
- GarageBand (macOS)

## Temporary Solution

For testing, you can:
1. Download a free notification sound from the sites above
2. Use a simple "ding" or "chime" sound
3. Convert to MP3 if needed using online converters

## Recommended Sounds

- Gentle chime
- Two-tone notification
- Soft bell sound
- Pleasant "ding"
